# Mobile-App-Redesign

This repository contains a UI/UX redesign of the Zomato mobile application.  
The main goal of this project is to improve the visual aesthetics, usability, and user experience of the original app.

---

## 📱 Project Overview

The redesign focuses on:
- Clean and modern UI
- Improved navigation flow
- Attractive color palette and typography
- Enhanced user interaction experience

---

## 🛠️ Tools Used

- **Lunacy** – For UI design
- **Figma** – For wireframes and prototypes
- **Canva / PowerPoint** – For presentation
- **GitHub** – For version control and sharing
